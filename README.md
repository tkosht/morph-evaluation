# morph-evaluation
Evaluation for Morphological Analysis Tools like MeCab, Janome, SudachiPy, nagisa

# Install

## install python libraries

```bash
sh bin/pip.sh
```

## Install Dictionaries

### for MeCab

```
sh bin/install_mecabdic.sh
```

### for SudachiPy

```
bin/install_sudachidic.sh
```


# run

```bash
python classify_ldcc.py
```
